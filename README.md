# py-bot

Профессиональный Telegram бот на Python с использованием фреймворка aiogram.

## Описание

py-bot - это Telegram бот, предоставляющий базовые функции взаимодействия с пользователями, включая:
- Приветственное сообщение при старте
- Функцию эхо-повтора сообщений
- Возможность получения случайных изображений собак

Проект построен с использованием современных практик разработки на Python и поддерживает контейнеризацию через Docker.

## Особенности

- **Асинхронная архитектура**: Использование aiogram для эффективной обработки сообщений
- **Модульная структура**: Разделение кода на сервисы и конфигурации
- **Docker поддержка**: Готовые конфигурации для развертывания в контейнерах
- **Простая расширяемость**: Легко добавлять новые команды и функциональность

## Требования

- Python 3.10+
- Poetry для управления зависимостями
- Docker (опционально, для контейнеризации)

## Установка

### Локальная установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/kavlan-dev/py-bot.git
   cd py-bot
   ```

2. Установите зависимости:
   ```bash
   poetry install
   ```

3. Создайте файл `.env` в корне проекта и добавьте токен вашего Telegram бота:
   ```
   TOKEN=ваш_телеграм_бот_токен
   ```

4. Запустите бота:
   ```bash
   poetry run python src/py_bot/main.py
   ```

### Установка с Docker

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/kavlan-dev/py-bot.git
   cd py-bot
   ```

2. Создайте файл `.env` с токеном бота:
   ```
   TOKEN=ваш_телеграм_бот_токен
   ```

3. Соберите и запустите контейнер:
   ```bash
   docker-compose up --build
   ```

## Использование

После запуска бота, вы можете взаимодействовать с ним через Telegram:

- `/start` - Начать взаимодействие с ботом
- `/dog` - Получить случайное изображение собаки
- Любое текстовое сообщение - Бот повторит ваше сообщение (эхо-функция)

## Конфигурация

Бот использует переменные окружения для конфигурации:

- `TOKEN` - Обязательный токен Telegram бота

Конфигурация загружается из файла `.env` или переменных окружения.

## Разработка

Для разработки и тестирования:

1. Установите зависимости для разработки:
   ```bash
   poetry install --with dev
   ```

2. Запустите тесты:
   ```bash
   poetry run pytest
   ```
